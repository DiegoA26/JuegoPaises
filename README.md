# 🌍 Juego de Países

Aplicación web interactiva donde el usuario pone a prueba sus conocimientos sobre países y sus poblaciones. El objetivo es adivinar correctamente y conseguir la mayor puntuación posible.

---

## 🚀 Características

* Mostrar información de países
* Sistema de puntuación
* Feedback inmediato al usuario
* Interfaz sencilla y dinámica

---

## 🛠️ Tecnologías usadas

* HTML5
* CSS3
* JavaScript (Vanilla)

---

## 📦 Instalación

1. Clona el repositorio:

```bash
git clone https://github.com/tu-usuario/tu-repo.git
```

2. Entra en la carpeta del proyecto:

```bash
cd tu-repo
```

3. Abre el archivo `index.html` en tu navegador.

---

## ▶️ Uso

* Inicia el juego desde la pantalla principal
* Intenta adivinar correctamente si el segundo país tiene mas o menos poblacion
* Acumula puntos según tus aciertos
* Al finalizar, verás tu puntuación total
